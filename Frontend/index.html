🚀 About This Project

This is a Multi-Modal AI Content Marketing Engine built as part of the Infotact Solutions GenAI Internship Program.

It uses local AI (Ollama + phi3) to generate platform-optimized marketing content including blog posts, tweets, Instagram captions, SEO metadata, and AI-generated campaign images — all from a single brief.

🛠️ Technology Stack

🐍 FastAPI
High-performance async Python API
🦙 Ollama + phi3
Local AI — no API key needed
🎨 Pollinations.ai
Free AI image generation
⚡ HTML / CSS / JS
Responsive dashboard frontend
